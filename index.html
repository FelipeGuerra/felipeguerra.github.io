Breton

VTEX e-commerce project, developed under gulp task runner with Sass + bootstrap. Developed from scratch an wishlist where clients can create a list of products, this list also appears on costumer account and also the costumer can manage the list.

UniFTEC

VTEX e-commerce distance education Project. Developed an feature that some courses show a entrance exam. Another feature is an integration with clients REST API to check costumer document in order to apply a discount, also using the same API to costumers register.

VTEX e-commerce project, developed from scratch a feature to exchanges and product changes, where costumer can select the product inside the order, in orders page, wich the costumer can also select how, where and when want to change the product. Another feature is a store locator, using google maps API, where the costumer can finde the nearest store by chosen location.

OficinaFu

VTEX e-commerce project. Developed from scratch a feature to display an average of 50 diferent product colors, where the costumer can chose and buy more than one color of the same product as easy as possible, the feature uses the VTEX Rest API

VTEX e-commerce project. This project has a feature, made from scratch, that costumer can send a picture and choose some backgroud, after that a design team make a custom art, that can be approved by the costumer, all integrated with the platform API using REST. Another cool feature is the wheel of fortune optin, the costumer always gets some discount.
